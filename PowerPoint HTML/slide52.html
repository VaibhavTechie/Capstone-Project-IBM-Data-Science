Classification Accuracy

This chart visualizes the built model accuracy for all classification models used in the analysis. The accuracy of each model is compared to determine which model has the highest classification accuracy.

Key Findings:

The bar chart shows the accuracy of Logistic Regression, SVM, Decision Tree, and KNN models. The model with the highest accuracy can be identified from the chart.

  • Logistic Regression Accuracy: 84.64%
  • SVM Accuracy: 84.82%
  • Decision Tree Accuracy: 87.50%
  • KNN Accuracy: 84.82%

Conclusion:

Among the tested models, the Decision Tree model exhibited the highest classification accuracy at 87.50%, followed by SVM and KNN, both at approximately 84.82%. Logistic Regression also performed well with an accuracy of 84.64%. These results suggest that the Decision Tree model is the most effective for this dataset.
