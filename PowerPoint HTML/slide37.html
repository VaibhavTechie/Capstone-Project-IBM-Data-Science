%sql SELECT Booster_Version FROM SPACEXTABLE WHERE PAYLOAD_MASS__KG_ = (SELECT MAX(PAYLOAD_MASS__KG_) FROM SPACEXTABLE);

Purpose:

This query retrieves the booster version that carried the maximum payload mass from the SpaceX dataset.

Key Elements:

Query: This SQL query selects the Booster_Version from the SPACEXTABLE where the PAYLOAD_MASS__KG_ equals the maximum payload mass recorded in the table.

Result: Please see the result on the right side of the slide.

Observations:

The result shows the booster versions that carried the maximum payload mass.

Interpretation:

This query helps identify which booster versions were used for the heaviest payloads, indicating their capacity and reliability for large missions.

Conclusion:

Understanding which boosters carry the maximum payload is crucial for planning future missions and optimizing the use of high-capacity boosters.
